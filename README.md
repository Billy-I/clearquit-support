# ClearQuit Support Site

Public GitHub Pages site for ClearQuit support, privacy, and terms.

- Support: `support.html`
- Privacy: `privacy.html`
- Terms: `terms.html`

The site is static and does not include app source code, credentials, analytics, or cookies set by ClearQuit.

ClearQuit is local-first. Private iCloud sync is an optional ClearQuit Pro feature; the app uses Apple's private CloudKit database and does not require a ClearQuit account.
